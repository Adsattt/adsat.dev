# Hello World

Welcome to the new site! This is built with **Astro**, **MDX**, and **Tailwind CSS v4**.

## Why Astro?

I was bored with Next.js and wanted to experiment with something new. Astro ships **zero JavaScript** by default, which is perfect for a portfolio and blog.

## What's MDX?

MDX lets me write **Markdown** with **JSX components** embedded. Like this code block:

```typescript
const greeting = "Hello from Astro!";
console.log(greeting);
```
